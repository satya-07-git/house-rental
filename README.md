# 🏠 RentEase – Rental Home Manager

A modern, responsive web application to manage tenants across **18 bachelor rooms** in a rental home.

## ✨ Features

- **18-Room Dashboard** — Visual grid showing occupancy status of all rooms at a glance
- **Tenant Details** — Collect and store Name, Phone, Aadhar/ID, Occupation, Move-in Date, Rent, Emergency Contact, Email, and Notes
- **Bill Payment Indicator** — One-click toggle between ✅ Paid / ❌ Unpaid, editable anytime by the owner
- **Live Stats** — Header shows real-time counts of Occupied, Vacant, Paid, and Unpaid rooms
- **Search & Filter** — Instantly search by name, room number, phone, or occupation. Filter by All / Occupied / Vacant / Bill Due
- **Data Persistence** — All data saved to browser `localStorage` — no server or database needed
- **Responsive** — Works on desktop, tablet, and mobile

## 🚀 How to Use

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Click a room card to add or edit a tenant
4. Click the **Paid / Unpaid** pill on any card to toggle the bill status instantly
